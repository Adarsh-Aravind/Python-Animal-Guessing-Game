# Python Animal Guessing Game 🐾

A simple command-line guessing game written in Python where the player tries to guess a randomly selected animal from a list within a limited number of attempts.

This project was created as a beginner Python exercise to practice basic programming concepts such as loops, conditionals, lists, and random number generation.

---

## How the Game Works

1. The program randomly selects an animal from a predefined list.
2. The player is given **5 attempts** to guess the correct animal.
3. After each incorrect guess, the remaining attempts are shown.
4. If the player guesses correctly, they win the game.
5. If all attempts are used, the correct answer is revealed.

---

## Technologies Used

* Python 3
* `random` module

---

## How to Run

1. Make sure Python is installed.
2. Clone the repository:

```
git clone https://github.com/Adarsh-Aravind/python-animal-guessing-game.git
```

3. Navigate to the project folder:

```
cd python-animal-guessing-game
```

4. Run the program:

```
python game.py
```
